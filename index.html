Desktop Setup

Run FortiFocus locally, configure admin access, or share a Windows build

The current app is a JavaFX desktop project. You can launch it with Maven, load admin credentials from .env or environment variables, and package a shareable Windows app-image with a bundled Java runtime.

Run

Launch with Maven

Use run-fortifocus.bat, run-fortifocus.ps1, or run mvn javafx:run from the project root to open the desktop app.

Java 17+ JavaFX 21 Maven launch
Admin

Configure admin access

Copy .env.example to .env or set FORTIFOCUS_ADMIN_USERNAME and FORTIFOCUS_ADMIN_PASSWORD in your environment before opening the admin sign-in flow.

.env Env vars Separate overlay
Share

Build a Windows app-image

Run build-shareable.bat to generate a FortiFocus app-image and zip. The packaged build includes its own Java runtime, so your friend does not need Java installed.

FortiFocus.exe Bundled runtime ZIP output
